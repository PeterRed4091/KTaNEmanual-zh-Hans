Foreign Exchange Rates

关于“外币汇率”模块

假如炸弹成为了证券经纪人……

为解除该模块，拆弹专家需要拥有一台能够连接至万维网的设备。

该模块有能力连接至网络并查询外币汇率的状态。模块上会有网格状排列的 3 行 3 列的按键，每一个都带有发光二极管。当发光二极管正按顺序闪烁时，不要按下该模块的任意按钮。

按键依照所在行被分组为：

顶部一行：基准货币的 ISO 4217 字母代码。[1][2]
中间一行：目标货币的 ISO 4217 字母代码。[1][2]
底部一行：需要转换的货币金额。

所有发光二极管变为绿色：

将下列统一资源定位符输入你的可连网设备：
http://api.fixer.io/latest?base=XXX&symbols=YYY
把 XXX 替换为基准货币的 ISO 4217 字母代码。
把 YYY 替换为目标货币的 ISO 4217 字母代码。
你将收到以 JavaScript 对象表示法为格式的数据；寻找 {"YYY": 数字}，其中“数字”即是汇率。利用该汇率，将底部一行的数字转换至目标货币并向下取整，记下从左数第 2 个数字[4]。按下第 N 个按键，其中 N 即是记下的数字[5]（数按键时从上到下、从左到右）。

所有发光二极管变为红色：

在这种情况下，该模块未能查询到今日货币汇率。取得目标货币的 ISO 4217 数字代码，并记下从右数第 2 个数字。按下第 N 个按键，其中 N 即是记下的数字[5]（数按键时从上到下、从左到右）。

[1]：如果炸弹上有多于一节电池，基准货币代码与目标货币代码将互相替换。
[2]：这有可能改为 ISO 4217 数字代码。
[3]：注意货币汇率在欧洲中部时间下午 4 点左右会更新。
[4]：在转换后货币少于 10 的情况下，应记下的数字为 0。
[5]：在记下的数字为 0 的情况下，按下左上角的按键。
