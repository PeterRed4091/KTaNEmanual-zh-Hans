关于“方块定向”模块

即使这炸弹不会杀死我们，脑出血也可能致我们于死地。

为了解除炸弹的这一部分，你需要良好的三维定向技能。你需要利用底部的四个按键，将一个虚拟正方体旋转至指定的朝向。不幸的是该模块不会显示或表明该虚拟正方体当前的朝向，因此你不得不自己想象此正方体的状态。

左下角的两个按键会以俯瞰正方体的视角，分别顺时针或逆时针偏转该正方体。

右下角的两个按键会以虚拟观察者的视角，分别顺时针或逆时针翻滚该正方体。虚拟观察者的位置在模块上以眼睛表示。注：虚拟观察者的位置有可能改变。

例如，如果眼睛在底部，那么它正面向“前方”表面。按下“顺时针翻滚”会将“左侧”表面放置到“顶部”表面所在的位置。

［英汉对照］
“TOP”——“顶部”；“BOTTOM”——“底部”；“FRONT”——“前方”；
“BACK”——“后方”；“LEFT”——“左侧”；“RIGHT”——“右侧”

如果炸弹上的序列号包含字母 R：

旋转正方体，使得初始的左侧表面处于和初始的顶部表面相同的位置，然后按下 SET（设定）按钮。

否则，如果炸弹有带 TRN 标签的常亮指示灯，或者有带 CAR 标签的常亮／常灭指示灯：

旋转正方体，使得初始的底部表面处于和初始的右侧表面相同的位置，然后按下 SET 按钮。

否则，如果炸弹有 PS/2 接口，或者已得一振或更多振：

旋转正方体，使得初始的底部表面处于和初始的前方表面相同的位置，同时初始的左侧表面处于和初始的底部表面相同的位置，然后按下 SET 按钮。

否则，如果炸弹上的序列号包含数字 7 或 8：

旋转正方体，使得初始的右侧表面处于和初始的底部表面相同的位置，同时初始的后方表面处于和初始的前方表面相同的位置，然后按下 SET 按钮。

否则，如果炸弹上有多于两节电池，或者虚拟观察者的初始位置正面向初始的左侧表面：

旋转正方体，使得初始的顶部表面处于和初始的底部表面相同的位置，然后按下 SET 按钮。

否则：

旋转正方体，使得初始的顶部表面处于和初始的左侧表面相同的位置，然后按下 SET 按钮。

得一振时：

如果你得了一振，虚拟正方体会重置到初始位置。注意，如果观察者处于与之前不同的位置，你可能需要选一条新规则。
